# Virtualizer

The **Virtualizer** is a deprecated governance server.  The equivalent function
runs as an integration connector in the [Database Integrator OMIS](../../../integration-services/database-integrator)
that is part of the [Integration Daemon](integration-daemon.md) OMAG Server.

The Database Integrator OMIS is paired with the 
[Data Manager OMAS](../../../access-services/data-manager).
